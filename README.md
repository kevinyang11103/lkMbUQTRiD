# 基于协同过滤算法的音乐可视化系统 python911

## 项目概述

本项目是一款基于协同过滤算法的音乐可视化系统，主要利用 Python 语言开发，实现了音乐数据爬取、存储以及音乐推荐的可视化展示。

## 技术栈

- **后端**：Python
- **数据库**：MySQL
- **前端**：待定

## 功能模块

- **音乐数据爬取**：自动获取网络上的音乐及用户数据。
- **数据存储**：将爬取的数据存储至MySQL数据库中。
- **协同过滤算法**：根据用户历史行为进行音乐推荐。
- **可视化展示**：以图形化的方式展示音乐推荐结果。

## 系统角色

- **管理员**：负责系统的维护与数据管理。
- **用户**：通过系统发现和欣赏音乐。

## 运行环境

- **操作系统**：兼容Windows、Linux、MacOS。
- **Python版本**：Python 3.6 或以上。
- **依赖库**：requests, beautifulsoup4, pymysql 等。

## 部署步骤

1. 安装 Python 环境。
2. 安装所需依赖库。
3. 配置 MySQL 数据库。
4. 运行爬虫程序进行数据初始化。
5. 启动后端服务。
6. 可选：集成前端页面。
